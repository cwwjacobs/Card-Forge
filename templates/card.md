# Card Template

Use this shorter template when cutting or forging a Card quickly.

```yaml
card_id: ""
name: ""
type: ""
status: "candidate"
input: ""
move: ""
output: ""
done_when: ""
stop_if: ""
receipt: ""
permissions: "read_only"
```

A Card is valid when it tells an agent what to do, what to use, what to produce, when to stop, and what to record.
