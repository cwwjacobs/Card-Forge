# Prompts

Prompt stubs for using Card Forge Core.

These prompts are intentionally small. They help humans or agents draw, cut, forge, stack, build, run, and receipt Cards and Decks without turning the public core into a premium product pack.

## Included stubs

- `draw_cards.prompt.md`
- `cut_card.prompt.md`
- `forge_card.prompt.md`
- `build_deck.prompt.md`
- `run_deck.prompt.md`
