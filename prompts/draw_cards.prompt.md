# Draw Cards Prompt

Use this when you need to choose useful existing Cards for a task.

## Prompt

Given the task, source material, and available Card Registry, identify 3 to 7 relevant Cards to place in the current Hand.

Return:

- selected Card IDs,
- why each Card was drawn,
- Cards left on the Bench,
- any missing Card candidates.

Do not invent canon Cards. If a needed Card does not exist, mark it as a candidate.
