# Cut Card Prompt

Use this when messy source material contains a reusable workflow move.

## Prompt

Review the provided source material and extract candidate Cards.

For each candidate, return:

- proposed Card name,
- guessed Card type,
- input,
- move,
- output,
- done condition,
- stop condition,
- receipt expectation,
- source note.

Keep candidates small and bounded. Do not mark candidates as canon.
