# Tiny Build Notes

Before publishing an AI-built app, inspect available source files, check for missing context, and avoid unsupported claims.

## Source inventory

List all source files, config files, prompts, and missing assets.

## Dependency check

Check package files and imports for missing or risky dependencies.

## Claim boundary

Do not claim the app is secure, bug-free, compliant, production-ready, or fully tested unless the evidence proves it.

## Final receipt

Record what was checked, what was skipped, what is blocked, and the recommended next action.
