# Build Deck Prompt

Use this when Cards and Card Stacks need to become a Deck.

## Prompt

Given a goal, Card Registry, and available Stack files, assemble a draft Deck.

Return:

- Deck ID,
- Deck name,
- purpose,
- stacks in order,
- allowed permissions,
- run outputs,
- final gate,
- draw pool,
- claim boundary.

Do not include premium Deck Pack content in the public core unless explicitly approved.
