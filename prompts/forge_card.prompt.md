# Forge Card Prompt

Use this when a candidate Card needs to be cleaned into the standard Card shape.

## Prompt

Convert the Card candidate into a valid Card using the Card Forge template.

Return:

- Card ID,
- name,
- type,
- status,
- input,
- move,
- output,
- done when,
- stop if,
- receipt,
- permissions.

If the candidate is too vague, mark it as rejected or needs revision instead of forcing it into canon.
