# Claim Boundary

Card Forge is a reusable workflow system for structuring AI-assisted work into Cards, Card Stacks, Decks, Runs, and Receipts.

## Allowed claims

Card Forge may claim that it helps users:

- break messy work into smaller reusable workflow moves
- organize Cards into Card Stacks and Decks
- run Decks against real source material
- produce Receipts that record what happened
- make workflows easier for smaller or edge agents to follow

## Disallowed claims

Card Forge must not claim that it guarantees:

- correctness
- security
- compliance
- production readiness
- bug-free code
- safe deployment
- legal, financial, medical, or professional approval

## Readiness language

Prefer bounded language:

- `ready for review`
- `no blocker found by this Deck`
- `pass with warnings`
- `needs source`
- `needs human decision`

Avoid absolute language:

- `guaranteed safe`
- `production secure`
- `fully verified`
- `bug-free`
- `compliance approved`

## Human authority

Card Forge can structure and receipt work. It does not replace human judgment, review, or responsibility.
