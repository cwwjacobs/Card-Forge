# Card Index

This is the human-readable index of known Cards.

The source of truth is `registry/card_registry.jsonl`.

## Starter candidates

| Card ID | Name | Type | Status | Use |
|---|---|---|---|---|
| `check.source_inventory.v0_1` | Source Inventory | Check Card | candidate | Start a Run by listing and classifying source material. |
| `contract.no_overclaim.v0_1` | No Overclaim | Contract Card | candidate | Prevent unsupported readiness, safety, or production claims. |
| `gate.human_approval.v0_1` | Human Approval Gate | Gate Card | candidate | Pause before risky actions or canon changes. |

## Status levels

- `candidate`: may become a Card
- `cut`: extracted as a rough Card
- `forged`: cleaned into valid Card shape
- `canon`: approved reusable owned Card
- `deprecated`: replaced or retired
- `rejected`: explicitly not useful, unsafe, or duplicate
